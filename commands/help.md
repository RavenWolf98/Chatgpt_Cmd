# /help - Command List

Input:
 /help

Description:
List all available commands and what each command does.

Rules:
- Use commands/index.md as the source of truth for valid commands
- Include /help in the list
- Show command name and a short description for each

Output:
result only (plain text, no JSON)

Format:
Available Commands:

/help - List all available commands and what each command does
/calc - Evaluate a math expression
/convert - Convert a value from one unit to another
/t - Convert 24-hour time to 12-hour format
/ts - Calculate elapsed time between two times (or from a time to now)

- One command per line
- Do not wrap in JSON
- Do not include any other text
